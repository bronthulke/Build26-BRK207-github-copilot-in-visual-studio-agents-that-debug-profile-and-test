# GitHub Copilot in Visual Studio: Agents That Debug, Profile, and Test

This breakout session explores how GitHub Copilot in Visual Studio helps professional developers work more effectively across debugging, profiling, testing, and broader development workflows.

## What this session covers

In this session, you will see how agent-powered experiences in Visual Studio can help you:

- investigate and resolve issues more efficiently during debugging
- analyze performance bottlenecks using profiling workflows
- improve confidence and speed when working with tests
- accelerate common development tasks with deeper GitHub Copilot integration
- explore upcoming capabilities such as modernization, skills, merge conflict assistance, bring your own key support, and the GitHub Copilot SDK

## Getting started

To continue exploring after the session:

1. Download Visual Studio 2026 Insiders from visualstudio.com/insiders
2. Clone this repository
3. Review the README for session outcomes, technologies, and next steps
4. Use GitHub Copilot in Visual Studio to experiment with debugging, profiling, and testing workflows in your own projects

## Who this is for

This content is intended for professional developers who want to understand how GitHub Copilot in Visual Studio can support day-to-day development workflows, especially in complex C# and C++ environments.
